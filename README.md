# NETFLIX CONTENT STRATEGY ANALYSIS

This project explores Netflix's content strategy using python to analyze how differen types of content (movies vs TV shows),release timing, and genres contribute to viewership performance. 


## Project Objective

* Understand how Netflix allocates its content (shows vs movies) and how that drives viewership.
* Analyze seasonal and monthly trennds in content release and viewership.
* Identify which content types, languages or gernes perform best in terms of hours viewed.
* Extract data-driven insights that can perform content-strategy decisions for streaming platforms.


## Dataset Overview 

This project uses a Netflix content dataset for 2023, which includes:
* Content titles and types (movie ot TV shows).
* Genre, language, and release details (date, month, day of the week).
* Viewership metrics such as "Hours Viewed", used to quantify audience engagement.

## Tools and Libraries

* Python
* Pandas - for data cleaning, transformation and aggregation
* Plotly Express / Plotly GraphObjects -  for interactive visualizations
* Jupyter Notebook for exploration and reproducibility

## Key Analysis 

1. Content Type Analysis
     * Compare total viewership hours between Movies and TV Shows.
     * Visualize the distribution using bar charts to see which content type dominates.
2. Monthly Viewership Trends
     * Aggregate viewership by release month and content type.
     * Plot line charts to study how viewership shifts across months and content types.
3. Data Cleaning & Preprocessing
     * Clean the "Hours Viewed" column and convert it into a numeric format.
     * Handle missing values and prepare features for deeper analysis.
4. Strategic Insights
     * Identify peak months and content formats with the highest engagement.
     * Suggest data-backed recommendations for content release and library management.
